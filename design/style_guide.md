# Style Guide

## Typography
- Heading: Inter, 700, 2.5rem
- Body: Inter, 400, 1rem
- Small: Inter, 400, 0.875rem

## Colors
- Primary: `#3B82F6`
- Accent: `#FCD34D`
- Background: `#FFFFFF`
- Muted: `#F3F4F6`

## Spacing
- Padding: `px-4 py-2` for buttons
- Margin: `mt-6` for sections

## Component Rules
- Cards should always have `rounded-xl`, `shadow-md`
- Buttons use `rounded-full`, `hover:scale-105`
